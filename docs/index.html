Lo esencial de WhatsApp, con atención a la accesibilidad

La aplicación utiliza XMPP como transporte, pero presenta una experiencia centrada en conversaciones de WhatsApp y en el uso cotidiano desde Windows.

Conversaciones completas

Lee y envía mensajes, responde, edita, reacciona, reenvía y consulta estados de entrega y lectura.

Grupos y menciones

Participa en grupos, identifica remitentes y utiliza autocompletado accesible para menciones.

Multimedia local

Reproduce notas de voz y video, abre documentos y consulta imágenes o stickers descargados de forma segura.

Notificaciones de Windows

Responde o marca como leído desde notificaciones, con controles para ocultar el contenido del mensaje.

Teclado y lector de pantalla

Foco predecible, etiquetas claras, navegación con flechas, Escape y Enter, y soporte adicional para NVDA.

Historial disponible

Conserva una caché local por cuenta para mostrar conversaciones conocidas mientras se sincroniza el servidor.

Cómo funciona el acceso

  1. Solicita una invitación

    La beta incorpora pocas personas a la vez para poder acompañar cada alta.

  2. Recibe tu cuenta

    Cada persona utiliza credenciales XMPP y una sesión del puente separadas.

  3. Vincula WhatsApp

    El cliente guía la vinculación y comienza a sincronizar chats después de autorizarla.

Privacidad explicada por capas

Los ajustes, credenciales protegidas por Windows, mensajes en caché y descargas se guardan localmente. Para prestar el servicio, el puente y el servidor XMPP también procesan información de la cuenta, mensajes, adjuntos y metadatos.

No presentamos el puente como cifrado de extremo a extremo entre WhatsApp y este cliente: el contenido debe ser procesado para trasladarlo entre ambos sistemas.
